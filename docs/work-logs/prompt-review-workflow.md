# 프롬프트 검토 작업 기록

## 설명

사용자는 프로젝트 루트에 `PROMPT_REVIEW.md` 파일을 만들고, AI에게 코드 요청 프롬프트를 다시 제공하여 적절성을 평가하고 개선하도록 요청할 기준을 정리하라고 지시했다.

이번 작업에서는 프롬프트 검토 기준 문서와 관련 작업 기록을 분리했다.

## 핵심 정리

작업한 내용은 다음과 같다.

1. 루트 경로에 `PROMPT_REVIEW.md`를 작성했다.
2. AI 코드 요청 프롬프트를 검토할 8가지 기준을 정리했다.
3. AI에게 다시 제공할 수 있는 검토 요청 프롬프트 템플릿을 추가했다.
4. 항목별 평가 방식과 개선된 최종 프롬프트 작성 원칙을 정리했다.
5. 관련 작업 기록을 `docs/work-logs/prompt-review-workflow.md`에 분리했다.

관련 문서는 다음과 같다.

```text
PROMPT_REVIEW.md
docs/work-logs/prompt-review-workflow.md
```

다음에는 실제 C++ 코드 요청 프롬프트 예시를 작성한 뒤 `PROMPT_REVIEW.md` 기준으로 평가하고 개선하는 예시를 추가할 수 있다.
