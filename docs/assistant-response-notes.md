# Assistant 답변 정리 문서

## 설명

이 문서는 프로젝트 초기 설정 과정에서 assistant가 응답한 주요 내용과 생성한 문서를 정리한다.

앞으로 assistant의 답변 중 프로젝트 규칙, 학습 내용, 문서화가 필요한 내용은 `docs` 폴더에 별도 문서로 저장하거나 이 문서에 요약해서 기록한다.

## 예제 코드

```cpp
// 이 문서는 답변 기록 정리용 문서이므로 실행용 C++ 코드는 포함하지 않는다.
```

## 핵심 정리

### 1. 문서 정리 폴더 생성

요청 내용:

```text
이 문서를 정리하기 위한 폴더 생성해줘
```

정리 내용:

- 문서 저장용 폴더로 `docs`를 생성했다.
- 프로젝트 문서들은 기본적으로 `docs` 폴더에 저장한다.

관련 경로:

```text
docs/
```

### 2. Assistant 응답 지침 작성

요청 내용:

```text
AGENTS파일 만들어서 위의 내용 적용해줘
```

정리 내용:

- 루트 경로에 `AGENTS.md`를 생성했다.
- assistant 답변 구조를 Markdown 학습 노트 형식으로 고정했다.
- C++ 코드가 필요한 경우 `cpp` 코드 블록을 사용하도록 규칙을 추가했다.
- 불필요한 인사말을 생략하도록 명시했다.

관련 문서:

```text
AGENTS.md
```

### 3. 문서 기본 언어 규칙 추가

요청 내용:

```text
모든 문서 파일들은 한글로 작성하는걸 기본으로 하되, 필요할 경우 영어로 해줘
```

정리 내용:

- 모든 문서 파일은 한글로 작성하는 것을 기본값으로 정했다.
- 기술적 정확성, API 이름, 에러 메시지, 공식 용어가 필요한 경우 영어를 허용했다.
- 사용자가 영어를 요청한 경우에도 영어를 사용할 수 있도록 했다.

관련 문서:

```text
AGENTS.md
```

### 4. 프로젝트 코딩 컨벤션 문서 작성

요청 내용:

```text
언리얼엔진의 코딩 스타일을 따라가는 컨벤션을 베이스로 하자
관련 내용 문서화해줘
```

정리 내용:

- Unreal Engine C++ 코딩 스타일을 기반으로 프로젝트 컨벤션 문서를 작성했다.
- 파일 이름, 타입 이름, Boolean 변수, Enum, 클래스 작성 순서, `const` 사용, 주석 규칙을 정리했다.
- Unreal 전용 규칙은 일반 C++ 학습 프로젝트에 맞게 조정했다.

관련 문서:

```text
docs/project-conventions.md
```

### 5. 커밋 메시지 컨벤션 문서 작성

요청 내용:

```text
커밋 할 때 커밋 제목과 내용에 대한 컨벤션 문서가 필요해
관련 내용 작성해서 정리해줘
```

정리 내용:

- 커밋 제목과 본문 작성 규칙을 문서화했다.
- Conventional Commits 스타일을 참고하되 프로젝트에 맞게 단순화했다.
- `feat`, `fix`, `docs`, `refactor`, `test`, `chore` 등의 타입을 정리했다.
- 좋은 커밋 메시지와 나쁜 커밋 메시지 예시를 추가했다.

관련 문서:

```text
docs/commit-conventions.md
```

### 6. AGENTS 파일 한글화

요청 내용:

```text
에이전트 파일 한글로 수정해줘
```

정리 내용:

- `AGENTS.md`의 영어 설명을 한글로 수정했다.
- 답변 구조, 언어 규칙, C++ 코드 작성 형식을 한글 기준으로 정리했다.

관련 문서:

```text
AGENTS.md
```

### 7. 앞으로의 답변 정리 방식

앞으로 assistant 답변을 문서에 정리할 때는 다음 기준을 따른다.

1. 프로젝트 규칙과 관련된 답변은 `docs/*-conventions.md` 형식으로 정리한다.
2. C++ 학습 내용은 `docs/cpp-*` 형식으로 정리한다.
3. Git 작업 흐름은 `docs/git-*` 형식으로 정리한다.
4. 단순 작업 기록은 이 문서에 요약한다.
5. 문서에는 기본적으로 한글을 사용한다.

현재 문서 목록:

| 문서 | 목적 |
|---|---|
| `AGENTS.md` | assistant 응답 지침 |
| `docs/project-conventions.md` | 프로젝트 코딩 컨벤션 |
| `docs/commit-conventions.md` | 커밋 메시지 컨벤션 |
| `docs/assistant-response-notes.md` | assistant 답변 정리 |
