# AI-Learn 프로젝트 요약

## 설명

`AI-Learn`은 C++ 학습을 목표로, 로컬 환경에서 동작하는 AI 기반 콘솔 프로그램을 만들어 가는 저장소이다. 핵심 방향은 단순히 문법 예제를 모으는 것이 아니라, 실제로 실행 가능한 작은 프로그램을 구현하면서 C++의 클래스 설계, 파일 분리, 문자열 처리, 콘솔 입출력, 외부 프로세스 연동, 로컬 LLM 호출 구조를 함께 익히는 것이다.

현재 저장소에는 두 개의 주요 C++ 프로젝트가 있다.

1. `LocalAIChat`
   - Ollama의 `qwen3:8b` 모델을 사용하는 로컬 AI 채팅 콘솔 프로그램이다.
   - 사용자가 콘솔에 메시지를 입력하면 로컬 LLM에 요청을 보내고 assistant 응답을 출력한다.
   - 프로그램 실행 시 Ollama 서버가 꺼져 있으면 자동으로 시작하고, 프로그램 종료 시 실행 중인 `ollama.exe` 서버 프로세스를 함께 종료하는 흐름을 포함한다.

2. `TextMUDGame`
   - C++로 구현한 텍스트 기반 MUD 게임 프로젝트이다.
   - `look`, `go`, `items`, `take`, `use`, `inventory`, `talk`, `ask`, `save`, `load`, `help`, `quit` 같은 명령어를 처리한다.
   - 5차 구현 기준으로 `낡은 성문의 비밀` 시나리오를 사용하며, `ask` 명령어에서는 로컬 Ollama 모델을 통해 NPC 질문 응답을 시도한다.
   - Ollama 서버나 모델을 사용할 수 없는 경우에도 게임이 종료되지 않도록 fallback 응답을 제공한다.

이 프로젝트는 Visual Studio 솔루션 파일을 기준으로 빌드한다. CMake 기반 프로젝트가 아니며, 각 프로그램은 별도의 `.sln` 파일을 가진다.

```text
LocalAIChat\LocalAIChat.sln
TextMUDGame\TextMUDGame.sln
```

권장 빌드 구성은 다음과 같다.

```text
Debug | x64
```

문서화도 프로젝트의 중요한 일부이다. `docs` 폴더에는 프로젝트 기준, 코딩 컨벤션, assistant 작업 방식, 텍스트 MUD 설계 문서처럼 계속 참조할 문서가 저장된다. 개별 작업 기록은 `docs/work-logs/` 폴더에 따로 저장하여, 운영 규칙 문서와 작업 이력을 분리한다.

코딩 스타일은 Unreal Engine C++ 스타일을 참고하지만, Unreal 전용 매크로나 빌드 규칙을 강제하지 않는다. 학습용 C++ 코드로 읽기 쉬운 구조를 만드는 데 초점을 둔다. 예를 들어 클래스와 함수는 PascalCase를 사용하고, boolean 변수에는 `b` 접두사를 붙이는 규칙을 따른다.

전체적으로 `AI-Learn`은 다음 흐름으로 발전하는 프로젝트이다.

1. C++ 콘솔 프로그램의 기본 구조를 만든다.
2. 클래스를 역할별로 분리하며 프로젝트 구조를 익힌다.
3. 로컬 LLM인 Ollama와 연결해 AI 응답 기능을 붙인다.
4. 텍스트 게임이나 채팅 assistant처럼 실제 사용 가능한 작은 프로그램을 만든다.
5. 구현 과정과 설계 의도를 문서로 남겨 학습 노트로 재사용한다.

## 핵심 정리

- `AI-Learn`은 C++ 학습을 위한 로컬 AI 콘솔 프로그램 저장소이다.
- 주요 프로젝트는 `LocalAIChat`과 `TextMUDGame`이다.
- `LocalAIChat`은 Ollama `qwen3:8b` 모델을 사용하는 로컬 AI 채팅 프로그램이다.
- `TextMUDGame`은 명령어 기반 텍스트 MUD 게임이며, NPC 질문 응답에 로컬 LLM을 활용한다.
- 빌드는 Visual Studio `.sln` 파일 기준으로 진행하며, 권장 구성은 `Debug | x64`이다.
- 문서는 기본적으로 한글 Markdown으로 작성하고, C++ 예제는 `cpp` 코드 블록을 사용한다.
- `docs` 루트에는 계속 참조할 기준 문서를 두고, 개별 작업 기록은 `docs/work-logs/`에 저장한다.
- 프로젝트의 장기 목표는 C++ 문법 학습을 넘어, 구조화된 로컬 AI assistant와 AI 기반 텍스트 게임을 직접 구현하며 학습하는 것이다.
